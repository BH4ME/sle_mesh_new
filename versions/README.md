# Versions

This directory records repository, firmware, test, and hardware version evidence.

## Current Versions

- Latest repository record: [v4.4.190](./v4.4.190/VERSION.md)
- Current firmware version: [v4.4.190](./v4.4.190/VERSION.md)
- Current enclosure version: [hardware/enclosures/sle-pcb-enclosure/v1.1.4](../hardware/enclosures/sle-pcb-enclosure/v1.1.4/)

## Version Rules

- Repository record versions track documentation, layout, scripts, and hardware-material changes.
- Firmware versions track behavior changes that affect the WS63 firmware or flash package.
- Hardware versions track PCB, schematic, enclosure, and manufacturing-file changes.
- Each version directory should include at least `VERSION.md`; broader changes may also include `MANIFEST.md`.
- Historical version directories preserve their evidence unless a safety or encoding fix is needed.
