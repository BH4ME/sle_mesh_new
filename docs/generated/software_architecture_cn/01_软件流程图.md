# 01 软件流程图

本节描述当前 `v4.5.46-minimal` 软件从启动到运行的主流程。系统核心由两个主要任务组成：

- `TeamNetworkTask`：负责组网、SLE 收发、HTTP、CLI、GPS、状态 LED、核心状态机 tick。
- `TeamDisplayTask`：负责 ST7789/LVGL 初始化、状态页面刷新、事件页面刷新。

## 软件启动流程

```mermaid
flowchart TD
    Boot["固件启动"] --> Entry["team_network_entry"]
    Entry --> SpawnDisplay["创建 TeamDisplayTask"]
    Entry --> SpawnNetwork["创建 TeamNetworkTask"]

    SpawnDisplay --> DisplayInit["初始化 ST7789 / LVGL"]
    DisplayInit --> DisplayLoop["显示循环: 读取快照 -> 刷新状态/事件 -> tick"]

    SpawnNetwork --> Bootstrap["team_network_task_bootstrap"]
    Bootstrap --> LedInit["初始化 WS2812 状态灯"]
    Bootstrap --> GpsInit["初始化 GPS/NMEA"]
    Bootstrap --> Identity["读取 MAC 并生成 route_id/self_label"]
    Bootstrap --> HttpStart["启动 SoftAP HTTP 服务"]
    Bootstrap --> UartCli["启动 UART CLI"]
    Bootstrap --> NvRestore["读取 NV 保存的角色配置"]
    NvRestore --> RoleConfigured{"是否已有角色配置?"}

    RoleConfigured -- 否 --> Idle["保持未配置状态, 等待 Web/CLI 配置"]
    RoleConfigured -- 是 --> ConfigureRole["配置 leader 或 member 角色"]
    ConfigureRole --> SleStart["启动 SLE client/server"]
    SleStart --> MainLoop["进入 TeamNetworkTask 主循环"]
    Idle --> MainLoop
```

## 主循环流程

```mermaid
flowchart TD
    Loop["TeamNetworkTask 每 10ms 循环"] --> Cli["处理 CLI 队列"]
    Cli --> ClientTick["SLE client tick"]
    ClientTick --> Configured{"角色已配置?"}

    Configured -- 否 --> Sleep["sleep 10ms"]
    Configured -- 是 --> NodeTick["sle_team_node_tick"]

    NodeTick --> Role{"当前角色"}
    Role -- Leader --> LeaderScan["更新 leader 扫描策略"]
    LeaderScan --> RelayOpt["每 8s 尝试稳定态 relay 优化"]
    RelayOpt --> RelayClient["检查 relay client 是否需要启动/暂停"]

    Role -- Member --> RelayClient

    RelayClient --> GpsTick["GPS 定位上报 tick"]
    GpsTick --> LedTick["WS2812 状态刷新"]
    LedTick --> DisplayPub["发布显示状态快照"]
    DisplayPub --> Sleep
```

## 收包处理流程

```mermaid
flowchart TD
    Rx["SLE 收到数据"] --> DecodeMesh["解码 Mesh Packet"]
    DecodeMesh --> Unwrap["解 GROUP_DATA: channel_hash / cipher_mac / app_payload"]
    Unwrap --> DecodeApp["解码 App Packet"]
    DecodeApp --> Check["校验 team_id / channel_hash / leader_term"]
    Check --> NeedForward{"是否不是发给自己?"}

    NeedForward -- 是 --> Forward["relay 节点转发"]
    NeedForward -- 否 --> Dispatch["按 app_msg_type 分发"]

    Dispatch --> Hello["HELLO: 入网/重入网"]
    Dispatch --> Ack["ACK: 确认策略下发"]
    Dispatch --> Config["CONFIG: 更新心跳/relay 参数"]
    Dispatch --> Route["ROUTE_UPDATE: 更新 parent/next_hop"]
    Dispatch --> Heartbeat["HEARTBEAT: 更新 last_seen/RSSI/在线状态"]
    Dispatch --> Pos["POS_REPORT: 更新位置"]
    Dispatch --> Alert["ALERT: 离队/超时/低电量等告警"]
```

## 发送流程

```mermaid
flowchart TD
    Business["业务消息: HELLO/HEARTBEAT/POS/ALERT/CONFIG/ACK/ROUTE_UPDATE"] --> AppBuild["编码 App Packet"]
    AppBuild --> GroupWrap["封装 GROUP_DATA"]
    GroupWrap --> MeshWrap["封装 Mesh Packet"]
    MeshWrap --> Role{"发送节点角色"}

    Role -- Leader --> FindConn["按 dst_id / next_hop 找 SLE client 连接"]
    Role -- Member --> Upstream["通过 server 连接发给 parent/leader"]
    Role -- Relay --> Downstream["relay 可通过 client 连接发给 child"]

    FindConn --> SleSend["WS63 SLE 发送"]
    Upstream --> SleSend
    Downstream --> SleSend
```

## 状态显示流程

```mermaid
flowchart TD
    NetworkTask["TeamNetworkTask"] --> PublishStatus["发布 role/self/online/offline/event_count"]
    NetworkTask --> PublishEvent["发布 join/rejoin/lost/left 事件"]
    PublishStatus --> Shared["共享显示快照"]
    PublishEvent --> Shared
    Shared --> DisplayTask["TeamDisplayTask"]
    DisplayTask --> ShowStatus["ws63_st7789_show_status"]
    DisplayTask --> ShowEvent["ws63_st7789_show_event"]
    DisplayTask --> Tick["ws63_st7789_tick"]
    Tick --> Panel["ST7789 屏幕"]
```
