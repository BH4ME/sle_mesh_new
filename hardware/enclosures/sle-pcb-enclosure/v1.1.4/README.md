# SLE PCB Enclosure v1.1.4

## 版本摘要

- 移除 GPS 区域上方的贯穿窗口。
- GPS 区域改为 0.8 mm 薄顶皮，外观封闭，同时给天线区域保留较薄材料。
- 显示窗口到 GPS 区域之间保留约 2.05 mm 连接桥。
- 继承 v1.1.3 的可打印配合改进：PCB/屏幕间隙加宽、电池高度降低、滑盖加厚、顶部螺丝孔和 M1 螺母/沉孔扩大。
