# CAD Assets

This directory contains STL files copied from the active CAD/model tree.

- `stl/hardware/...` contains the current hardware enclosure STL files that also live near the hardware documentation.
- `stl/models/...` contains historical and variant STL models from the local model tree.

Large STEP/GLB simulation exports are intentionally excluded from Git. Re-add them through release assets or Git LFS if they become required for mechanical handoff.
