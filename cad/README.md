# CAD Assets

This directory contains the STL files kept for the handoff repository.

- `stl/models/...` contains only the latest three versioned enclosure model sets:
  - `sle_enclosure_v2.1.17_no_screen_top_lock`
  - `sle_enclosure_v2.1.18_no_screen_top_lock_harder`
  - `sle_enclosure_v2.1.19_screen_top_lock`

Large STEP/GLB simulation exports are intentionally excluded from Git. Re-add them through release assets or Git LFS if they become required for mechanical handoff.
